# GameOfLife

## How to play

## Dependencies

-Termbox2

## Build Project

1. Navigate to the project root directory

2. Create a build directory and navigate to it:

```shell
mkdir build
cd build
```

3. Run CMake and build the project:

```shell
cmake ..
make
```

## Run Project

```shell
./GameOfLife
```

## Run Unit Tests

```shell
./tests/unit_tests
```
